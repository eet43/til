# 문자열과 변수 함께 출력하기

## 변수가 문자열일 때
변수가 문자열이라면 편하게 + 를 사용하여 묶어 출력하면 된다. <br><br>

```python 
value = "detail"
print("detail : " + value)
```

## 변수가 문자열이 아닐 때
변수가 문자열이 아닌 그 외 변수라면 ( 정수, 리스트, 딕셔너리 등 ) , 를 사용하여 묶어 출력해야 한다. 이거 때문에 1시간 삽질했다 <br><br>
```python 
value = {
  "name" : name,
  "city" : city,
}
print("detail : ", value)
```
